# GoldenHour AI

GoldenHour AI is an AI-powered emergency healthcare coordination platform that connects ambulances and hospitals in real time.

## Features
- Real-time patient vitals assessment
- Hospital readiness monitoring
- AI-powered hospital recommendation
- Emergency alert system
- Live hospital capacity tracking

## Tech Stack
- React.js
- Vite
- JavaScript
- CSS
- HTML
